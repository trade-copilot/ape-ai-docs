### Order Rejected: Insufficient Buying Power

**Cause:** Not enough cash for the trade

**Solution:**
1. Check Portfolio â†’ Buying Power
2. Reduce quantity (1 contract instead of 2)

### Can't Set Stop Loss

**Cause:** Some brokers don't support stop loss on options via API

**Solution:**
1. Set price alert instead at $2.80
2. When alert hits, manually close position from chat
3. **Say: "Sell my AMD call"** â†’ Execute immediately

**Cause:** Market moved between clicking and filling

**Solution:**
- Use limit orders (not market)
- Set limit at mid-point of AI's range
- Example: AI says $3.50-$3.70, set limit $3.60
